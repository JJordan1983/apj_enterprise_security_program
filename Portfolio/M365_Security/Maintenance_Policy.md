# Maintenance Policy  
APJ SecureCloud  
Version 1.0

## 1. Purpose
Defines requirements for SaaS-layer system maintenance in Azure Gov.

## 2. Scope
Covers:
- Application updates
- Key rotation
- Config updates
- Library updates
- CI/CD pipeline updates

Azure Gov host maintenance is inherited.

## 3. Requirements
- Maintenance must be scheduled
- Changes must go through CM process
- Emergency maintenance requires IR Lead notification
- Logs must be retained

## 4. Remote Maintenance
Remote access must use:
- MFA
- CA policy enforcement
- PIM elevation

## 5. Approval
Approved by APJ Security Program Lead.

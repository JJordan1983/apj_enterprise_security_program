# **IR-9 – Information Spillage**

Not applicable for APJ SecureCloud  
because:

- No customer content or PHI/PII is processed  
- Only metadata flows into the system  
- No stored customer data exists that can be “spilled”  

If metadata exposure occurs:

- Treated as a security incident  
- Investigated and reported according to IR-6

# **IR-10 – Integrated Incident Response Team**

APJ SecureCloud’s IR team includes:

- Security Officer (IR Lead)  
- Cloud Security Analyst  
- DevOps Engineer  
- Privacy Liaison  
- Microsoft Support (when escalated)  
- FedRAMP assessor inputs (during audits)  

Team receives integrated alerts through:

- Sentinel  
- Defender for Cloud  
- Azure Service Health  
- Email notifications

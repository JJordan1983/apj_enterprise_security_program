# Media Protection Policy  
APJ SecureCloud  
Version 1.0

## 1. Purpose
Ensures all digital media is protected from unauthorized access.

## 2. Applicability
Since APJ SecureCloud is metadata-only and cloud-native:
- No removable media is used
- No data is stored on local devices
- No physical media is created

## 3. Cloud Media Protection
All system metadata resides in:
- Azure Storage
- Log Analytics
- Sentinel

Protected by:
- RBAC
- Encryption at rest
- Key Vault
- Conditional Access

## 4. Disposal
Deletion is performed via:
- Secure Azure Storage deletion
- Retention policies

Approved by APJ Security Program Lead.

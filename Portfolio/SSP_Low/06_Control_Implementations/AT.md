# 06. Control Implementations – AT: Awareness & Training  
APJ SecureCloud – FedRAMP Low (LI-SaaS)

APJ SecureCloud maintains a formal cybersecurity training and awareness program for all personnel with access to corporate or production resources. Because the system boundary relies heavily on Azure Government inherited controls, training aligns to FedRAMP, DoD Cyber Awareness, and Microsoft cloud operations best practices.

---

## **AT-1 – Awareness & Training Policy and Procedures**

APJ SecureCloud maintains documented security awareness and training policies, reviewed annually by the Security Officer.  
Policies include:

- Security Awareness Policy  
- Role-Based Training Requirements  
- Incident Reporting Responsibilities  
- Annual Acknowledgement Requirements  
- Disciplinary process for training non-compliance  

Procedures define:
- How training assignments are issued  
- Tracking and reporting through LMS  
- Escalation for overdue training  
- Storage of certificates (encrypted repository)  

Policy location:  
`Portfolio/M365_Security/Policies/Security_Awareness_Policy.md`

---

## **AT-2 – Literacy, Awareness, and Training**

All APJ personnel with system access must complete training before obtaining access to:

- Azure Gov IL2 tenant  
- Production environment  
- PIM elevation  
- Sensitive operational logs  
- Customer metadata ingestion pipelines  

Training modules include:

### **Annual Cybersecurity Awareness Training**
- Phishing recognition  
- Password management  
- MFA best practices  
- Handling sensitive metadata  
- Cloud security basics  

### **Quarterly Micro-Training**
- Updates on cloud security threats  
- Microsoft Defender / Sentinel alerts  
- Social engineering trends  
- Excessive permissions risks  

### **Delivery**
- LMS automated assignment  
- Completion tracked by Security Officer  
- Certificates stored for 6 years for audit readiness  

No APJ operator may access production resources until AT-2 completion is verified.

---

## **AT-3 – Role-Based Security Training**

Certain APJ roles require additional specialized training:

### **Cloud Operations / Engineering Staff**
- Azure Admin & Security fundamentals  
- PIM elevation safety  
- Key Vault access procedures  
- Secure pipeline operations  
- Azure DevOps & GitHub sec
